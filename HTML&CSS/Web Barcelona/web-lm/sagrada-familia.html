Sagrada Família

La Sagrada Família és una basílica catòlica dissenyada per l'arquitecte Antoni Gaudí. És una de les obres arquitectòniques més impressionants del món i un símbol de Barcelona.

Les seves torres altes, les façanes decorades amb escultures i la seva arquitectura única la fan una visita obligada per a qualsevol persona que vingui a la ciutat.
